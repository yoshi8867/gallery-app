# 갤러리 앱 요구사항 명세서

**작성일:** 2026-04-10  
**버전:** 1.0  
**작성자:** 요구사항 분석 에이전트

---

## 1. 앱 개요

| 항목 | 내용 |
|------|------|
| 앱 이름 | 갤러리 앱 (Gallery App) |
| 플랫폼 | Android |
| 목적 | 기기 내 사진/동영상을 조회·관리하는 단독 기기용 갤러리 앱 |
| 미디어 소스 | MediaStore API (로컬 저장소만 사용, 서버 없음) |
| 인증 | 없음 (단독 기기 전용, 로그인 불필요) |

### 앱 설명

기기에 저장된 사진과 동영상을 빠르고 직관적으로 탐색할 수 있는 안드로이드 갤러리 앱이다. Material Design 3을 기반으로 한 현대적인 UI를 제공하며, 앨범별 분류, 즐겨찾기, 검색, 공유 등 핵심 기능과 함께 슬라이드쇼 기반 스토리 기능을 지원한다.

---

## 2. 주요 기능 목록

### 기능 우선순위 분류

| 우선순위 | 기능 | 설명 |
|----------|------|------|
| P1 (필수) | 사진/동영상 보기 및 관리 | 갤러리 기본 기능. 미디어 목록 조회, 전체화면 보기, 삭제, 정보 확인 |
| P1 (필수) | 앨범/폴더별 분류 | 폴더 단위 앨범 목록, 앨범 내 미디어 그리드 보기 |
| P2 (중요) | 즐겨찾기 기능 | 미디어에 즐겨찾기 마킹, 즐겨찾기 목록 화면 |
| P2 (중요) | 공유 기능 | 선택한 사진/동영상을 SNS, 카카오톡 등 외부 앱으로 공유 |
| P2 (중요) | 사진 검색 | 날짜, 태그 등 조건으로 미디어 검색 |
| P3 (선택) | 스토리 기능 | 슬라이드쇼 형태 자동재생, 최후순위 구현 |

## 3. 화면 목록 및 설명

### 3.1 하단 내비게이션 구조

NavigationBar + NavigationBarItem 컴포넌트를 사용하여 주요 섹션을 탭으로 전환한다.

| 탭 | 화면명 | 설명 |
|----|--------|------|
| 사진 | 사진 메인 스크린 | 기기 내 모든 사진/동영상을 날짜 기준 그룹화하여 표시 |
| 앨범 | 앨범 리스트 스크린 | 폴더별 앨범 목록 표시 |
| 스토리 | 스토리 리스트 스크린 | 스토리 목록 표시 |
| 부가 메뉴 | 메뉴 모달 창 | 다음의 버튼들 나열: 동영상 리스트 스크린, 즐겨찾기 리스트 스크린, 최근 항목 리스트 스크린, 위치별 스크린, 공유 앨범, 사진첩 정리 스크린, 휴지통 스크린, 설정 |

---

### 3.2 화면별 상세 설명

#### 사진 메인 스크린 (Photo Main Screen)
- **컴포넌트:** LargeTopAppBar (Collapsing) + LazyColumn + StickyHeader
- **기능:**
  - 날짜별 섹션 그룹화 (StickyHeader로 날짜 고정 표시)
  - LazyVerticalGrid 형태로 썸네일 표시
  - 핀치 줌으로 그리드 열 수 조절 (1.5단/3단/4단/7단/11단/20단:7단 이하에서는 실제 썸네일 item들을 배치해서 선택했을 때 사진 스크린으로 진입할 수 있게 하고, 11단부터는 실제 item이 아님. 보이기만 썸네일 묶음으로 보이고, 11단에서 선택 시 7단으로 핀치 아웃한 효과. 마찬가지로 20단에서 선택하면 11단으로 핀치 아웃한 효과.)
  - 동영상 항목에 재생시간 배지 표시(좌하단 반투명)
  - 상단 검색 바 접근(돋보기 아이콘)
  - 다중 선택 모드 (공유, 삭제, 즐겨찾기 일괄 처리)

#### 앨범 리스트 스크린 (Album List Screen)
- **컴포넌트:** LazyColumn + LazyVerticalGrid (1단(목록형)/2단/3단/ 전환)
- **기능:**
  - 일반 폴더 앨범 목록
  - 앨범 커버 이미지 + 앨범명 + 미디어 개수 표시
  - 앨범 정렬 옵션 (이름순, 날짜순, 개수순)

#### 앨범 뷰 스크린 (Album View Screen)
- **컴포넌트:** LazyVerticalGrid + AnimatedVisibility (서랍식 사이드 패널)
- **기능:**
  - 선택한 앨범 내 미디어 그리드 표시(서랍이 안 열린 것 기준으로 1.5단/3단/4단/7단/12단. 총 5종. 하지만 서랍이 열리면 공간이 더 좁으므로 1.5단/2단/3단/5단/9단. 총 5종. -> 이중 1.5단은 사진 가로형과 세로형이 섞여있을 때 자연스럽게 섞어 배치하는 것으로 정확한 격자의 그리드는 아님. 5종 중 마지막 단계에서는 선택 시 핀치 아웃 효과가 있는 것이고, 실제 아이템이 선택되어 사진 뷰 스크린으로 넘어가지지는 않음.)
  - 서랍(Drawer) 형태의 앨범 전환 패널
  - 정렬 및 필터 옵션
  - 다중 선택 모드

#### 사진 뷰 스크린 (Photo View Screen)
- **컴포넌트:** HorizontalPager + LazyRow(하단 썸네일) + AnchoredDraggable(상세정보 패널)
- **기능:**
  - 좌우 스와이프로 사진 전환 (HorizontalPager)
  - 하단 썸네일 스트립 (LazyRow)
  - 핀치 줌 및 팬 제스처
  - 위로 드래그시 상세정보 패널 확장 (AnchoredDraggable)
    - 파일명, 날짜, 크기, 해상도, 위치 정보 표시(파란 글씨로 '편집'이라고 된 텍스트가 있어서 클릭하면 편집 가능)
  - 하단 얄은 바: 즐겨찾기(하트 아이콘), 편집(연필 아이콘: 실제 기능은 안 함), AI(AI 아이콘. 기능 안 함.), 공유(공유 아이콘), 삭제(휴지통)
  - 동영상인 경우 ExoPlayer 재생 컨트롤, 뮤트 버튼

#### 특수 앨범 스크린

| 스크린 | 설명 |
|--------|------|
| 동영상 스크린 | 기기 내 모든 동영상만 필터링하여 표시 |
| 즐겨찾기 스크린 | 즐겨찾기 마킹된 미디어 목록 (앨범 뷰와 동일 레이아웃) |
| 최근 항목 스크린 | 최근 추가된 미디어 (30일 또는 전체 기간 필터) |
| 휴지통 스크린 | 삭제된 미디어 (30일 보관, 복원 또는 영구 삭제 가능) |

#### 위치 스크린 (Location Screen)
- **기능:**
  - GPS 메타데이터가 있는 미디어를 국가/도시별로 ㄴ분류
  - 지역별 대표 이미지 카드 표시
  - 지역 선택 시 해당 미디어 목록으로 이동

#### 지도 스크린 (Map Screen)
- **컴포넌트:** GoogleMap + MarkerComposable
- **기능:**
  - 지도 위에 미디어 위치 마커 클러스터링 표시
  - 마커 탭 시 해당 미디어 미리보기
  - 지도 이동/줌에 따라 마커 동적 업데이트

#### 검색 스크린 (Search Screen)
- **기능:**
  - 파일명 검색
  - 위치 기반 검색 (도시명 등)
  - 미디어 타입 필터 (사진만, 동영상만)
  - 검색 결과 그리드 표시

#### 스토리 리스트 스크린 (Story List Screen) - P3
- **기능:**
  - 날짜/앨범 기반 자동 생성 스토리 목록
  - 스토리 커버 이미지 + 제목 카드
  - 스토리 커버 이미지는 계속 이동함(오른쪽으로 흐름)

#### 스토리 뷰 스크린 (Story View Screen) - P3
- **기능:**
  - 슬라이드쇼 형태 자동재생
  - 재생 속도 조절
  - 일시정지/재생 제어
  - 배경 음악: 음원 파일 넣어두고 모두 음원으로만 재생

| 분류 | 기술 |

| 디자인 시스템 | Material Design 3 |

## 5. 비기능 요구사항

### 5.1 성능
- 앱 초기 실행 후 갤러리 로드 시간 2초 이내 목표
- 썸네일 지연 로딩으로 스크롤 성능 확보 (60fps 유지)
- 대용량 미디어 파일도 끊김 없이 재생 (ExoPlayer 버퍼링 전략 적용)

### 5.2 UX
- Material Design 3 가이드라인 준수
- 다크 모드 / 라이트 모드 지원
- 접근성 (콘텐츠 설명, 최소 터치 타겟 48dp 이상)
- 가로 모드 지원 (사진 뷰, 동영상 뷰)

### 5.3 권한
- 미디어 읽기 권한: `READ_MEDIA_IMAGES`, `READ_MEDIA_VIDEO` (Android 13+) 또는 `READ_EXTERNAL_STORAGE` (Android 12 이하)
- 미디어 쓰기 권한: `WRITE_EXTERNAL_STORAGE` (삭제, 수정 시)
- 위치 정보 권한: `ACCESS_FINE_LOCATION` (지도/위치 스크린)
- 권한 미허용 시 안내 화면 제공

### 5.4 호환성
- 최소 지원 버전: Android 10 (API 29)
- 목표 지원 버전: Android 14 (API 34)

### 5.5 데이터
- 서버 없음, 모든 미디어는 기기 로컬 저장소에서 MediaStore API로 조회
- 즐겨찾기, 태그, 앱 설정 등 앱 자체 데이터는 Room DB에 저장

## 6. 구현 우선순위 로드맵

### Phase 1 - 핵심 기능 (P1)
1. 프로젝트 기본 설정 (Hilt, Navigation, Room 등)
2. MediaStore API 연동 및 미디어 로드
3. 사진 메인 스크린 (날짜별 그룹, 핀치 줌)
4. 앨범 리스트 스크린
5. 앨범 뷰 스크린
6. 사진 뷰 스크린 (HorizontalPager, 상세정보 패널)
7. 동영상 재생 (ExoPlayer)
8. 특수 앨범 (최근 항목, 동영상, 휴지통)

### Phase 2 - 부가 기능 (P2)
9. 즐겨찾기 기능 (마킹, 즐겨찾기 스크린)
10. 공유 기능 (Intent 공유)
11. 검색 스크린 (날짜, 파일명, 타입 필터)
12. 위치 스크린 (GPS 메타데이터 파싱)
13. 지도 스크린 (Google Maps 연동)

### Phase 3 - 선택 기능 (P3, 최후순위)
14. 스토리 리스트 스크린
15. 스토리 뷰 스크린 (슬라이드쇼 자동재생)
