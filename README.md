# 갤러리

Android 갤러리 앱 — Kotlin + Jetpack Compose + Material Design 3

## 기술 스택

- **언어**: Kotlin
- **UI**: Jetpack Compose + Material Design 3
- **아키텍처**: MVVM (ViewModel + StateFlow + Repository)
- **미디어**: MediaStore API, Coil, ExoPlayer (Media3)
- **지도**: Google Maps SDK for Compose
- **DB**: Room (즐겨찾기)
- **DI**: Hilt
- **내비게이션**: Navigation Compose
