애니메이션 & 인터랙션 설명
핀치 줌 — 그리드 열 수 전환 (Photo Main / Album View)
두 손가락 핀치 인/아웃으로 그리드 열 수(1.5단 → 3단 → 4단 → 7단 → 11단 → 20단)가 전환됩니다. Composable에서 transformable + pointerInput을 조합하고, 열 수 변화는 animateIntAsState로 부드럽게 보간합니다. 11단/20단은 저해상도 썸네일 경량 렌더링으로 전환되며, 탭 시 핀치 아웃 효과만 적용됩니다.
LargeTopAppBar 접힘 (사진 메인 스크롤)
LazyColumn 스크롤 시 LargeTopAppBar가 SmallTopAppBar 높이로 축소됩니다. TopAppBarScrollBehavior.exitUntilCollapsed()를 사용하며, 제목 폰트 크기는 22sp → 16sp로 lerp() 보간됩니다. 검색 바는 스크롤 진행도에 따라 알파가 0으로 페이드 아웃됩니다.
앨범 뷰 서랍 패널 (Album View — 스와이프 제스처)
화면 오른쪽 가장자리에서 왼쪽으로 스와이프하면 앨범 전환 패널이 열립니다. AnimatedVisibility + slideInHorizontally / slideOutHorizontally 트랜지션을 사용합니다. 서랍이 열리면 그리드 영역이 좁아지며(1.5/2/3/5/9단) 배경에 scrim(반투명 오버레이)이 적용됩니다.
다중 선택 모드 — 롱프레스 진입
아이템 롱프레스 시 해당 셀이 scale(0.9)으로 축소되며 선택 체크 오버레이가 나타납니다. 상단 SelectionTopBar가 fadeIn + slideInVertically로 등장하고, 하단 NavigationBar는 fadeOut + slideOutVertically로 숨겨집니다. 선택 해제 시 역방향 애니메이션으로 복원됩니다.
사진 뷰 상세정보 패널 (AnchoredDraggable — 위로 드래그)
사진 뷰에서 위로 드래그하면 파일 정보 패널(파일명, 날짜, 크기, 해상도, 위치)이 올라옵니다. AnchoredDraggableState로 Hidden → HalfExpanded → Expanded 3단계 앵커를 관리하며, 드래그 오프셋에 따라 패널 높이가 실시간으로 변경됩니다. 배경 사진은 드래그 진행도만큼 어두워집니다.
HorizontalPager 사진 전환 (Photo View)
좌우 스와이프로 사진이 슬라이드 전환됩니다. 현재 페이지는 scale(1.0), 인접 페이지는 scale(0.95)로 약간 축소되어 깊이감을 줍니다. 스와이프 중 하단 액션 바와 썸네일 스트립은 alpha로 페이드 처리됩니다.
스토리 커버 이미지 자동 흐름 (Story List)
스토리 카드의 커버 이미지가 오른쪽 방향으로 자동 스크롤됩니다. rememberInfiniteTransition()으로 이미지 오프셋을 주기적으로 이동시키거나, LazyRow + animateScrollToItem()을 반복하여 무한 스크롤처럼 보이게 구현합니다.
